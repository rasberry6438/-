eraTW4.956用他者変身コマンド追加パッチ+α

・内容
　素質:変化 持ちのキャラクターに他者変身コマンド(COM358)を追加します
　素質の自動付与は行っていないので口上で追加して使ってください
　実質マミゾウ専用の素質ですが他キャラでもご利用いただけます
　
　同梱の口上についてはそちらのreadmeを参照してください
　
　@HPH_PRINTの修正(BPBを正しく変換できるように)
　
・詳細 注意点
　コマンドで選択されたキャラの画像リソースを元のキャラの状態を参照して表示しています
　メイン画像表示処理に多少手を加えています。予期せぬエラーが出る可能性があるのでバックアップの上ご利用ください
　デフォルトでは恋慕or愛欲時のみコマンドを表示します　
　見た目を変えるだけでそれ以外のイベント等は現在実装していません
