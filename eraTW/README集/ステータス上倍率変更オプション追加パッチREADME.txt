--- ステータス上昇倍率変更オプション追加パッチ Ver1.0 ---

概要: OPTIONに各ステータス(ABL)の上昇倍率を変更する項目を追加します

使い方: 素のeraTW バージョン4.920に同梱のERBを上書きしてください

注意事項: ゲーム性を著しく損なう可能性があるため、過度な利用はお勧めいたしません

改造方法: もっと倍率高くOR低くしたい方は、ABL_UP_DATA.ERHのMAX_ABL_UP_RAITとMIN_ABL_UP_RAITをいじってください。やりすぎると壊れます

免責事項: 当パッチを当てたことによるトラブル等は対応いたしかねますのでご了承ください

ライセンス: 改造等ご自由にしてください
