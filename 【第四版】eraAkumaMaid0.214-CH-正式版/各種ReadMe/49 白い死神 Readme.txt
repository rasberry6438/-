更新履歴

	2022/08/19　ver0.022
………半年に一回更新と二周年更新は守れた！でも、守れただけなんです。悲しい事に。
前回比にして約47KBしか増えてないので大規模更新を期待してくれていた希少な方にはごめんなさい。

さて、何時もの無駄な前振りは兎も角として今回の更新についてをば。
新規で書き下ろしたのは胸愛撫、口交、口に射精、撫摸股間の計四つです。
一応パターンはあるのでそれなりには見れると思います。
ただ、えっちかは……あんまり自信が無いです。それは何時もの事ですが。
後は、追記と言う形でTalkと、個別イベント：取り留めない幕間の日記を少しだけ。

今回の事について語るのは此処までにして、次の話をしましょうか。
次の予定は10月に千紗の半年更新を目指していく予定を立てています。
その次は新年の1月に後回しにしていた白色彼女についてのお話を書ければ良いなと思っています。
終わりを司る白色彼女の始まりの御伽噺を。
では、また次の更新で。

	2022/02/21　ver0.021
殆ど増えてません！
一部约会とエロだと手交と正常位やV挿入中に阴茎尺寸変更が増えたけれど、
それでも前回更新からの期間を考えれば進みは遅いの一言。その上、イベ日を遅刻する始末。
ペンネームの元ネタであるとっても遅筆家を晒す醜態となりました。

と、新規は少ないながらもTalkの方ががらりと変わりました。
此処で書いて説明するよりも実際にPLAYして見て貰った方が分かると思うので深くは語りませんが……
若しかしたら前の方が想像がかき立てられて良かったと言う声もあるかもですが、それはごめんなさい。
ちょこちょこと変わってない箇所が有ますが……多分見逃しかわざとです。
この様な場所ですが、dialogue_frameを設計したブリート桑には感謝を。

次は今年の4/4の千紗の更新でお会いしましょう。
…………今更ですが新年明けまして御目出度う御座います。
今年も何卒ポンコツ死神を宜しくお願いいたします。

	2021/08/19　ver0.02
面白色作品とは何なのか。単純に閲覧した人を感動させたモノだけを面白色と言うのか。
それの答えを恥ずかしい事ながら私は出したことが有ません。

何せ書いてる時も、書きあがった時も、これで本当に良いのだろうか。
と自身が書いたモノに思うのですから。

もちろん、万人受けするような作品を書けるとは一切思っておらず、
しかして、多くの人に面白色と思ってほしいと相反した思いが在るのも確か。
だけれども、結局誰かの目線を意識せずに書きたいと思っているのも有ます。

受けを期待して書いてるのか、それとも自分はこうだ、と叫びたくて書いてるのか。
自分の事が一番分からない。
在りたい姿と、そう在る姿是つだって乖離しているのですから。

絵に描いた餅を求める様なお話。果ての無いモノを追いかける無為さ。
これが私にとっての創作論、何て格好つけて言ってみます。


何て、戯言語りの前語りはそこそこに。
一年が早いと感じる反面、この一年は何やらバタバタと忙しかったので充実していたと感じる部分も有ました。
ただ過去の自分に文句を言うのであれば、千紗を書いてから四ヶ月後に新角色を公開するのじゃないよですね。
後は、それをすっかり忘れて6月に更新かけるのじゃないよ。
とっても遅筆家であることを自覚してるならそれに見合ったペース配分を考えようねと文句を言うのでした。

さて、今回増えたのは個別イベントと日常がメインです。
と言っても、どちらもとあるイベントを踏まないと殆ど増えないので轻轻地だけそのトリガーを。
保全作業と言う名の彼女が居睡眠中に「整理睡姿」と色々と増えます。
エロ要素が増えるわけじゃないのでエロを望んでいた人にはごめんなさい、と謝ります。
何処かポンコツな彼女がよりポンコツになる、そんなくだらないTalkが増えます。

一応エロも少しだけ増やしました。
秘部開帳、扒開阴道を少しだけ書きました。
私事ですがエロを書く度にとことんエロを書くのが苦手なんだなぁと痛感します。
なので、期待していないでしょうけれど期待しないでください。

では、そろそろ更新履歴の打招呼(?)はここまでに。
何のかんのとこれからも更新は頑張っていきたいので1mm程度の期待でお待ちください。
では、また次の更新で。
