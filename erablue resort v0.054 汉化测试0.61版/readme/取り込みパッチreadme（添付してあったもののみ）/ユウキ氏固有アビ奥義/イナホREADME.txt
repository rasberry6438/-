
□ERB/口上_キャラ個別ERB/1025_イナホ
	〇新規作成（アビリティ・奥義・固有特性・探索中口上）
		◆奥義：『天華繚乱・稲妻招雷』
			・12.5倍敵全体対象奥義/麻痺効果
			・Lv50：倍率15倍に強化/麻痺効果の持続ターン強化
		◆1アビ：『百鬼夜行』
			・味方全体にバリア効果・追撃効果
			・Lv30：消費MP減少
			・Lv50：バリア効果・追撃効果の性能アップ
		◆2アビ：『殺生陣』
			・敵全体に3回光1.5倍ダメージ/恐怖効果・感電効果（累積連撃率ダウン）
			・Lv40：消費MP減少
			・Lv60『夢幻・殺生陣』：倍率2倍に強化/与ダメ減少効果追加
		◆3アビ：『九尾の神速』
			・Lv40で習得
			・自身に九尾の神速効果（DA率・TA率アップ/高揚効果）
			・Lv80：効果量アップ/与ダメ上昇効果追加
		◆固有特性：『九尾天雷妖獣姫』
			・奥義発動時、味方全体の攻撃力が累積上昇/奥義ゲージが溜まりにくい
			・1番目に編成時、味方全体の攻撃力が上昇/味方が奥義を発動する度、「フィーバーゲージ」が上昇
		◆固有特性：『クロスブルーフリッパー：特殊』
			・通常攻撃・奥義発動時、「フィーバーゲージ」が上昇
			・「フィーバーゲージ」が100％のターン開始時、ゲージを全て消費して自身にフィーバー効果（必ずTA/ターン開始時、味方全体にランダムな強化効果）
			・TA時、パワーフリップ：特殊（対象に自属性ダメージ）が発動
			・連続でTA時、パワーフリップの性能アップ
				＊「フィーバーゲージ」について：通常攻撃は連撃数×5％、奥義は10％溜まります。ただしフィーバー効果中は溜まりません。
