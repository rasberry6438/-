;作成内容の概要：
;　1.ZPショップのEXキャラ開放処理の改修
;　　・キャラの簡易紹介を追加
;　　・リストに戻らず次のキャラを確認するボタンを追加
;　　・キャラ開放をキャンセルした際、表示ページが0ページに戻らないように変更
;　　・ZP無料回数を使い切った際、料金が全キャラ一律50ZPになる不具合を修正
;　　　※ 本パッチ適用前に作成したセーブデータでは修正されません
;　　・EXキャラがどんどん増えそうなのでリスト表示上限を200から500に拡張
;
;　2.画像フォルダ選択の改修
;　　・画像フォルダ選択ボタンの横に画像フォルダ数の表示を追加
;　　・画像フォルダの数が1個の場合でも画像が見れるように変更
;　　・ランダム表示時にデフォルト画像を優先して表示するように変更
;　　・ランダム表示時に同じ画像が表示されないように変更
;
