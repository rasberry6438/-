;パッチ名：地の文改変用パッチ
;作者：
;作成日：2024/12/23
;
;作成内容の概要：

;加筆したのはパッチとしてうｐしやすいけど削減したのはちょっとうｐし辛い
;でも更新のたびに一箇所ずつ書き換えるのも面倒くさいし……
;と思ったのでファイルを放り込めば解決するようにしたかった

;各種地の文を表示する際に自作関数を読み込むパッチです
;デフォルト関数の末尾に"_MOD"を加えた関数が存在する場合、代わりにそちらの内容を表示します


;現在使える関数
;・各種MESSAGE_COM, MESSAGE_COM_非口上時
;　これらの関数はMOD側の返り値に依らず元々の地の文を表示しない

;・各種PALAM_MESSAGE_
;・MESSAGE_えっちな罰ゲーム
;・SHOOT予告口上
;・SHOOT絶頂口上
;　これらの関数はMOD側の返り値が0の場合元々の地の文を表示する
