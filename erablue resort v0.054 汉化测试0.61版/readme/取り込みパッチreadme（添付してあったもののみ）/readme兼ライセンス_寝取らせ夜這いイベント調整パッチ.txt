;パッチ名：寝取らせ夜這いイベント調整パッチ
;作者：
;作成日：2025/01/27
;
;作成内容の概要：
;◆寝取らせ夜這いイベント関連
;　○寝取られ性癖持ちが複数いる場合に寝取らせイベント発生率が下がらないように調整
;　　・寝取られ性癖持ちが1組の場合の発生率は調整前と同じ。
;
;　○寝取らせイベントを断られたキャラはしばらくの間寝取らせイベントの発生率が下がる処理を追加
;　　・寝取られ性癖持ちを増やし過ぎて見たいカップルが全然来ないケースへの対策1。
;
;　○寝取らせイベントの候補が多いときは候補の中から選択する処理を追加
;　　・寝取られ性癖持ちを増やし過ぎて見たいカップルが全然来ないケースへの対策2。対策1では足りない場合用。
;　　・イベント候補が10組以上の時、5組をランダムに選出、その中から選択するようになります。
;
;　○寝取らせうふふ開始前に他のキャラを睡眠状態にするように変更
;　　・陥落済キャラが起きていると対象がダウンした際に呼び出す選択肢が出るため。
;
;◆ウソ告白関連
;　○ウソ告白の特殊イベントの発生率を増加、1日1回しか発生しないように変更
;　　・発生率が低すぎてなかなか見れなかったので。
;　　　断った後に選択が出ないようにする対策は回数制限で実施。
;
;　○ウソ告白の特殊イベントの恋慕陥落処理を調整
;　　・恋慕対象に登録し、恋慕レベルも上がるように
;　　・寝取られた側にも履歴を追加
;
;　○ウソ告白の特殊イベントに関係破棄の選択肢を追加
;　　・イベント発生時の寝取らせ主導キャラと献上品キャラが恋慕関係の場合のみ、関係を破棄させることが可能
;
;　○恋心度と性愛度がMAXになるとウソ告白の特殊イベントが発生しなくなる不具合を修正
;
;◆その他
;　○「性交渉をお願いする」のセフレになる際の処理を調整
;　　・性交渉した回数に「貸し出し回数」も影響するように変更
;　　・セフレ陥落する条件を満たしている場合は初回お願い時でも成功するように変更
;　　・セフレ陥落はお願い成功時のみ発生するように変更
;
;　○「浮気」を所持したキャラが妊娠/出産した際に履歴に登録される内容を「知り合い」から「浮気相手」に変更
;　　・知り合い表記は味気なかったので…ついでに寝取らせ関係ないですが近親だった場合の表記も追加
;　　　対象が陥落済の場合はそちらの表記が優先されます
;
