;パッチ名：画像ファイル名チェック機能追加パッチ
;作者：
;作成日：2024/11/02
;
;作成内容の概要：
;　画像ファイル名がフォーマットに沿った名前かどうかチェックする機能を追加
;　デバッグモードで起動時、タイトル画面にボタンが追加されます
;　同時に顔画像と短冊画像が揃っていないファイルも抽出します
;　(単純に画像を用意していないだけの可能性もありますが、誤記等で揃っていないケースもあるので)
;
;　キャラ固有の衣装や差分がある場合、チェックNGにならないようにヘッダファイルに定義を追加可能です
;
;　<注意点>
;　・通常プレイで使用する機能ではないためデバッグモード専用の機能になっています
;　・ファイル名が間違っているファイルをピックアップしますが、「どこがどう間違っているのか」までは判定できません
;　・フォルダ名が間違っている場合はチェックされません
;　・奥義ゲージMAX差分は短冊画像がなくてもエラー判定しません(顔画像しか使用しないため)
;
