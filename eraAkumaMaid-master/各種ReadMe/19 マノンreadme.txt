era悪魔で女仆　万音口上ライセンス

このライセンスは、era悪魔で女仆用　万音口上に適応されます。

・基礎的な規律はeraコミュニティ規則に準じます。
・era悪魔で女仆以外のeraヴァリアントへの、すべてあるいは一部のコード流用を禁止します。
・era悪魔で女仆で使用する目的による改変再配布は許可されます。
・基本的に追記及び改変自由ですが、万音の角色クター性を著しく損なう改変を禁止します。
・他角色クターと万音の関連性についての改変は関連角色との交渉結果に準じます。
・不具合の修正や最適化といった、era悪魔で女仆内での動作を損なわないための改変は許可されます。

本ライセンスは、同梱した万音口上バージョン及び、それ以前の万音口上バージョンに適応されます。
本ライセンスは発行された地点から有効になります。
本ライセンスの未読・既読の有無を問わず、万音口上を利用する際には本ライセンスへの義務を負います。

●CEVENTについて
・覚醒の有無は、CEVENT関数で「淫魔覚醒：万音」フラグを格子すればいけます。
・强行破瓜した場合「破瓜：万音を强奸する」、覚醒時に破瓜すると「許された喪失」、恋慕・相思相愛の場合「幸せな喪失」のフラグになります。
