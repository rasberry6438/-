【おまけ】一部キャラクターの下着関連のデータを変更。
	ブラジャーに関してはキャラ毎に設定されている肩書きの公式衣装の着用時を想定。
	公式イラストやGBVSでのブラ紐の有無、脇や背中を露出している衣装の際にバックベルトが見えているか否か、
	バージョン違いの衣装の際に身に着けているか否か等から、ブラを身に着ける人物か否かを考慮して主観で設定。

	パンツに関しては、設定された肩書きの衣装で似合いそうな物を選択。

	田舎娘や星晶獣等、世間に疎そうなキャラにはノーブラ／ノーパンも設定しています。


0.3：
【おまけ】ステータス画面から「ノーブラ」の表示を削除。
	思いのほか「ノーブラ」キャラが増えてしまい「ノーブラ」である事が珍しくない雰囲気になった為、
	着けている事が「当たり前」の世界ならノーブラは下品かも知れないが、
	着けていなくてもおかしくない環境なら敢えて明記する必要も無いのでは、と思い至り削除。
	きっと揺れても痛くない人種。ｸｰﾊﾟｰ靭帯の事は気にするな、きっと強靭で謎再生したりするんだ。


【おまけ】パイズリ系コマンドで「パイズリ経験」が上がる様に修正。
	コマンド「パイズリ」実行中に「胸愛撫」等のコマンドを実行しても「パイズリ経験」が上昇せずに「口淫経験」が上昇している事に疑問を覚えて修正。
	「パイズリフェラ」が別コマンドとして存在しているのならば「パイズリ」中に口は使っていないだろう、と言う想定も兼ねた修正になります。
	
	
【おまけ】ステータス「ブラ着用」「下着着用」の表示を「下着(上)」「下着(下)」に変更。
	公式イラストでさらしを身に着けているキャラが「ブラ着用」になっていた為「さらしキャラはさらしを身に着けていて欲しい」と思い、胸部に下着として身に着けているもの全般を意味しての「下着(上)」に変更。
	これでキャミソールやブラトップ等もカバー出来たのではと思います。
	
	それに伴い「下着着用」を「下着(下)」に併せて変更。


【おまけ】コマンド「ブラを脱がす」「下着を脱がす」の表示を「上の下着を脱がす」「下の下着を脱がす」に変更。
	上記の変更に合わせて変更。ツールチップも修正しています。
