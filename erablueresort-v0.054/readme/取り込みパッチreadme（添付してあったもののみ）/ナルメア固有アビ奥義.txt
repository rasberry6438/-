31_ナルメア\固有アビリティ_ナルメア.ERB
	ナルメアお姉ちゃんのアビリティです
	数値が多く自分の残念な頭ではレベルによるアビリティ成長が考えられなかった……
	・1アビ
		ターン消費無しで20MP消費して構え変更
		構えは 戦闘中カウント:戦闘行動キャラ:0 == 0 なら源氏,1なら神楽という感じです
		アビ発動後{アビリティ使用フラグ}に{現在ターン}を入れて {現在ターン} == {アビリティ使用フラグ} なら使用不可になる という形で同ターン中１回のみという処理を突っ込んであります
	・2アビ
		アビダメ効果と胡蝶相剋付与は共通、構えによって追加効果が変動する
		効果が変動するせいでMP消費が考えづらい
		4アビの胡蝶双舞で両方発動するので構えで変わる効果は関数化
	・3アビ
		一番構えによる変動がデカいのでMP消費本当にわからん…
		消費ＭＰ35は ディスペル ＋ 80％スロウ の値です 構えによって損するのも嫌ですよね
		これも胡蝶双舞対象なので処理を関数化
	・4アビ
		胡蝶双舞（構え効果が両方発動する）を付与して奥義ダメ上昇
		一応構え系列だと考え１アビと同じようにターン１処理を突っ込んであります
