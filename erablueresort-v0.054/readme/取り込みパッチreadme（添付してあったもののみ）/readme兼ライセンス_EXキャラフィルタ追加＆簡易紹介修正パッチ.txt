;パッチ名：EXキャラフィルタ追加＆簡易紹介/解説表示修正パッチ
;作者：
;作成日：2024/11/18;

;作成内容の概要：
;　キャラ解説のテンプレと口上エディタの出力を
;　「EXキャラフィルタ追加＆簡易紹介/解説表示修正パッチ」(eraRx4800)の変更に対応した形式に修正
;
;作成内容の概要：
;　○EXキャラの解放処理にフィルタ機能を追加
;　　・性別、画像の有無、口上の有無、固有アビ奥義の有無でフィルタ可能です
;
;　○キャラ簡易紹介の表示項目を追加/調整
;　　・Ａ処女表記と年齢表記を追加
;　　・キャラ解説の表示位置を右の空きスペースへ移動
;　　　(設定の「サイド領域非表示オプション」がONの場合は本文中に表示)
;
;　○キャラ解説の表示方法を調整
;　　・口上側の関数を直接呼ばずに表示処理関数を呼ぶように変更
;　　　口上側の関数にRETRUN忘れとか表示関数の実行忘れとかあったので、口上側はKSTRだけ設定すればいい形に
;　　・上記に合わせて各口上ファイルから不要になった処理を削除
;
