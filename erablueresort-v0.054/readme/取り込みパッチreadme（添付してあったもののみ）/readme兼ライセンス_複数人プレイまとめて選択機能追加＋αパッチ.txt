;パッチ名：複数人プレイまとめて選択機能追加＋αパッチ
;作者：
;作成日：2024/11/16
;
;作成内容の概要：
;　複数人プレイ関連の選択処理について、
;　対象が10人とかになると毎回選択するのが大変だったのでまとめて選択できるボタンを追加しました
;　加えて複数人プレイ関連の表示を一部調整しています
;
;　○呼び出すの対象選択にまとめて選択ボタンを追加
;　○同時モードの対象選択にまとめて選択ボタンを追加、ボタンに対応する情報の表示を追加
;　○同時モードの対象選択の選択キャラが1名でも決定できるように変更
;　　選択キャラ1名で決定すると同時モードではなく通常モードになります
;　○呼び出す, 同時モードの対象選択のボタン配置等を調整
;　○コスプレ衣装選択時に対象キャラの名前を表示するように変更
;
