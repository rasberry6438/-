;パッチ名：レポート強調表示機能追加パッチ
;作者：
;作成日：2024/11/29
;
;作成内容の概要：レポート強調表示の追加＋表示追加＋不具合修正
;○レポート強調表示の追加
;　一日の終了時のレポートについて、特定状況になった項目を強調表示します
;　強調する内容は以下の通り ※ ＋:ポジティブ系強調(青表示)/－:ネガティブ系強調(赤表示)　＋－同時に発生した場合は＋優先
;　<オナニー報告>
;　・強調項目なし
;
;　<夜這い報告>
;　＋夜這いに成功したキャラがいる
;
;　<縁結び報告>
;　＋オート縁結びが成立したキャラがいる
;
;　<性癖開拓報告>
;　＋性癖開拓に成功したキャラがいる
;　－性癖開拓できないキャラがいる
;
;　<従業員業務報告>
;　＋仕事レベルアップ
;　－知識習得/訓練のレベル上限到達
;
;　<妊娠報告>
;　＋妊娠/妊娠状態進行/出産
;
;　<リゾート収支計算>
;　－ZP不足によるフィールド解除
;
;　<リゾート人気変動>
;　－人気度ダウン
;
;　<依頼報告>
;　＋依頼達成
;
;○レポート表示の追加
;　・料理人レベルが上昇のした際の表示を追加
;　・訓練に既に最大レベルのキャラが就いている場合に経験値が手に入らない旨の表示を追加
;　・知識習得で上限までレベルが上がった際に自動的に仕事が解除される旨の表示を追加
;
;○不具合修正
;　・一部の常駐業務(※)がレベルアップ判定時に装飾品店のレベルを参照していた不具合を修正
;　　※ ロビー受付,バーテンダー,司書,プール監視員
;
